C A P S

Es una aplicación para mejorar la vida cotidiana de personas que requieren ayuda con tareas debido a impedimentos de salud, edad avanzada o sobrecarga de responsabilidades. Este sistema permite conectar a solicitantes de asistencia con asistentes locales de manera eficiente y segura, proporcionando una solución integral para la gestión de tareas diarias.
